## 사용 가이드

### 💬 카카오톡 발송 방법

1. **단체 발송**
   - 카카오톡 비즈니스 계정 사용
   - 또는 단체 채팅방 생성

2. **개별 발송**
   - 예약 확정 시: 예약자에게 개별 발송
   - 체크인 전날: 자동 리마인더
   - 체크아웃 당일: 감사 메시지

3. **예약 시스템 연동**
   - 예약 발생 시 자동 메시지 발송
   - 체크인 D-1 자동 안내
   - 체크아웃 당일 자동 안내

### 📝 메시지 커스터마이징

- {{예약자명}}: 실제 예약자 이름으로 변경
- {{객실명}}: 예약한 객실명으로 변경
- {{체크인날짜}}: 실제 체크인 날짜로 변경
- {{총금액}}: 실제 결제 금액으로 변경
- 연락처, 주소 등은 실제 정보로 변경

### 🎯 발송 타이밍

1. **예약 확정**: 예약 후 즉시
2. **입금 확인**: 입금 확인 후
3. **체크인 전날**: 체크인 1일 전 오후
4. **체크아웃 당일**: 오전 9시경
5. **이벤트**: 월 1회 정도
