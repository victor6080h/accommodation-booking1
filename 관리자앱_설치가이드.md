# 📱 속초아파트 관리자 앱 설치 가이드

---

## ✅ PWA(Progressive Web App) 완벽 지원!

속초아파트 관리자 시스템은 **PWA로 구현**되어 있어 앱처럼 설치하여 사용할 수 있습니다.

### **PWA 장점**
✅ 앱스토어 없이 바로 설치
✅ 홈 화면에 아이콘 추가
✅ 빠른 실행 속도
✅ 전체화면 모드 (브라우저 UI 숨김)
✅ 오프라인 지원 (일부 기능)
✅ 푸시 알림 지원 가능

---

## 📱 안드로이드 설치 방법

### **🌐 Chrome 브라우저 사용 (권장)**

#### **1단계: Chrome에서 관리자 페이지 접속**
```
https://accommodation-booking1.vercel.app/admin
```

#### **2단계: 로그인**
- 아이디: `admin`
- 비밀번호: `admin1234`

#### **3단계: 설치 배너 확인**
로그인 후 화면 하단에 **파란색 "앱 설치" 배너**가 나타남

![배너 예시]
┌─────────────────────────────────┐
│ 🏠 속초 아파트 관리자            │
│                                 │
│ [지금 설치]  [나중에]           │
└─────────────────────────────────┘

#### **4단계: 설치 버튼 클릭**
- **"지금 설치"** 버튼 클릭
- 또는 우측 상단 **⋮** (점 3개) → "홈 화면에 추가" 클릭

#### **5단계: 설치 확인**
팝업 창이 나타남:
```
속초 아파트 관리자를 설치하시겠습니까?

앱 이름: 속초 아파트 관리자
URL: accommodation-booking1.vercel.app

[취소]  [설치]
```

**"설치"** 버튼 클릭!

#### **6단계: 완료! ✅**
- 홈 화면에 **"속초관리"** 아이콘 생성
- 아이콘을 길게 눌러 원하는 위치로 이동 가능

---

### **📱 삼성 인터넷 브라우저 사용**

#### **1-2단계: 접속 및 로그인** (위와 동일)

#### **3단계: 메뉴 열기**
- 하단 **≡** (메뉴) 버튼 클릭

#### **4단계: 홈 화면에 추가**
- "페이지를 다음으로 추가" 선택
- "홈 화면" 선택

#### **5단계: 앱 이름 확인**
```
이름: 속초 아파트 관리자
```
- **"추가"** 클릭

#### **6단계: 완료! ✅**

#### **2단계: 로그인**

- 아래로 스크롤
- **"홈 화면에 추가"** 항목 찾기

![iOS 공유 메뉴]
```
복사
저장
즐겨찾기 추가
홈 화면에 추가 ← 이거!
북마크 추가
...
```

#### **5단계: 앱 이름 확인**
```
이름: 속초 아파트 관리자
URL: accommodation-booking1.vercel.app
```
- **"추가"** 버튼 클릭 (우측 상단)

#### **6단계: 완료! ✅**
- 홈 화면에 아이콘 생성
- 아이콘을 길게 눌러 위치 이동 가능

## 💻 PC (Windows/Mac) 설치 방법

### **🌐 Chrome 브라우저**

#### **1단계: 접속 및 로그인**
```
https://accommodation-booking1.vercel.app/admin
```

#### **2단계: 주소창 확인**
주소창 오른쪽에 **⊕** (설치) 아이콘 확인

![Chrome 주소창]
```
🔒 https://accommodation-booking1.vercel.app/admin  ⊕ ⋮
                                                    ↑
                                                  클릭!
```

#### **3단계: 설치 클릭**
팝업 창:
```
속초 아파트 관리자를 설치하시겠습니까?

[취소]  [설치]
```

#### **4단계: 완료! ✅**
- **Windows**: 
  - 바탕화면에 바로가기 생성
  - 시작 메뉴에도 추가됨
  
- **Mac**: 
  - Applications 폴더에 추가
  - Launchpad에서 확인 가능

---

### **🌐 Microsoft Edge**

#### **1단계: 접속 및 로그인** (위와 동일)

#### **2단계: 메뉴 열기**
- 우측 상단 **⋯** (점 3개) 클릭

#### **3단계: 앱 설치**
- "앱" → "이 사이트를 앱으로 설치" 클릭

#### **4단계: 완료! ✅**

## 🎯 설치 후 사용 방법

### **📱 모바일에서**

1. **홈 화면에서 아이콘 찾기**
   - 아이콘 이름: "속초관리" 또는 "속초 아파트 관리자"

2. **아이콘 클릭**
   - 앱처럼 전체화면으로 실행
   - 브라우저 주소창/메뉴바 숨겨짐

3. **자동 로그인**
   - 처음 로그인 후 상태 유지
   - 앱을 닫아도 로그인 유지

4. **바로가기 사용**
   - 아이콘을 길게 누르면 빠른 메뉴
   - "예약 확인" 또는 "객실 관리" 바로가기

### **💻 PC에서**

1. **바탕화면 또는 시작 메뉴에서 실행**

2. **독립 창으로 실행**
   - 브라우저와 별도 창
   - 전체화면 모드 지원

3. **작업 표시줄에 고정**
   - Windows: 작업 표시줄에 우클릭 → "고정"
   - Mac: Dock에 앱 추가

## 🔥 앱 설치의 장점

### **✅ 빠른 접근**
- 홈 화면 아이콘 클릭 → 바로 실행
- 브라우저 열고 URL 입력 불필요

### **✅ 전용 앱 느낌**
- 전체화면 모드
- 브라우저 UI 숨김
- 네이티브 앱처럼 동작

### **✅ 로그인 상태 유지**
- 한 번 로그인하면 계속 유지
- 매번 로그인 불필요

### **✅ 알림 지원 (향후)**
- 새 예약 알림
- 입금 확인 알림
- 체크인 시간 알림

### **✅ 오프라인 지원 (일부)**
- 인터넷 끊겨도 기본 UI 표시
- 캐시된 데이터 확인 가능

## 🔧 문제 해결

### **❓ 설치 버튼이 안 보여요**

**원인 1: 이미 설치됨**
- 홈 화면에 이미 아이콘이 있는지 확인
- 검색으로 "속초" 입력해보기

**원인 2: 브라우저 버전 낮음**
- Chrome: 최소 버전 80 이상
- Safari: iOS 14.3 이상
- 브라우저 업데이트 필요

**원인 3: 시크릿 모드**
- 시크릿/사생활 보호 모드에서는 설치 불가
- 일반 모드로 접속

**해결 방법**:
1. 브라우저 완전히 닫기
2. 다시 열고 재접속
3. 캐시 삭제 후 재시도

---

### **❓ 아이폰에서 "홈 화면에 추가" 없어요**

**원인**: Safari가 아닌 다른 브라우저 사용

**해결**:
1. Safari 브라우저로 전환
2. 관리자 페이지 재접속
3. 공유 메뉴(□↑) → "홈 화면에 추가"

---

### **❓ 설치했는데 아이콘이 안 보여요**

**해결 1: 홈 화면 검색**
- 여러 페이지 넘겨서 확인
- 검색으로 "속초" 입력

**해결 2: 앱 서랍 확인** (안드로이드)
- 앱 서랍(App Drawer)에서 확인
- "ㅅ" 항목에서 찾기

**해결 3: 재설치**
- 기존 아이콘 삭제
- 브라우저에서 재설치

---

### **❓ 설치 후 로그인이 계속 풀려요**

**원인**: 쿠키 설정 문제

**해결**:
1. **설정 확인**
   - Android: Chrome 설정 → 사이트 설정 → 쿠키 허용
   - iOS: 설정 → Safari → 모든 쿠키 차단 해제

2. **재로그인**
   - 로그인 시 "로그인 상태 유지" 체크
   - 브라우저 기록 삭제하지 않기

---

### **❓ PC에서 앱이 실행 안 돼요**

**해결 1: 바로가기 확인**
- 바탕화면 또는 시작 메뉴에서 찾기
- Windows: 검색으로 "속초" 입력

**해결 2: 재설치**
- Chrome에서 재설치
- Edge에서 재설치
