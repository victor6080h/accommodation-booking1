## 💰 환불 금액 계산기

### **빠른 환불액 계산 방법**

```
📅 체크인일: [YYYY-MM-DD]
📅 취소 요청일: [YYYY-MM-DD]
💵 예약 금액: [금액]원

→ 남은 일수: [체크인일 - 취소일] = X일

━━━━━━━━━━━━━━━━━━━━━
환불 계산
━━━━━━━━━━━━━━━━━━━━━

1️⃣ 환불율 확인
• 7일 이상 → 100%
• 5~6일 → 80%
• 3~4일 → 50%
• 1~2일 → 30%
• 당일 → 0%

2️⃣ 환불액 계산
예약금 × 환불율 = 환불액
[금액]원 × [X]% = [계산결과]원

3️⃣ 최종 환불액
환불액 - 수수료 = 최종
[계산결과]원 - 500원 = [최종금액]원
```

### **계산 예시**

#### **예시 1: 체크인 10일 전 취소**
```
예약금: 150,000원
남은 일수: 10일 (7일 이상)
환불율: 100%

계산:
150,000원 × 100% = 150,000원
150,000원 - 500원 = 149,500원

→ 최종 환불액: 149,500원
```

#### **예시 2: 체크인 5일 전 취소**
```
예약금: 100,000원
남은 일수: 5일 (5~6일)
환불율: 80%

계산:
100,000원 × 80% = 80,000원
80,000원 - 500원 = 79,500원

→ 최종 환불액: 79,500원
```

#### **예시 3: 체크인 3일 전 취소**
```
예약금: 120,000원
남은 일수: 3일 (3~4일)
환불율: 50%

계산:
120,000원 × 50% = 60,000원
60,000원 - 500원 = 59,500원

→ 최종 환불액: 59,500원
```

#### **예시 4: 체크인 1일 전 취소**
```
예약금: 200,000원
남은 일수: 1일 (1~2일)
환불율: 30%

계산:
200,000원 × 30% = 60,000원
60,000원 - 500원 = 59,500원

→ 최종 환불액: 59,500원
```

#### **예시 5: 체크인 당일 취소**
```
예약금: 180,000원
남은 일수: 0일 (당일)
환불율: 0%

계산:
180,000원 × 0% = 0원

→ 최종 환불액: 0원 (환불 불가)
```
