## 📋 수정 필요한 항목

### 예약 정보
- `[예약자명]` → 실제 예약자 이름
- `[연락처]` → 예약자 연락처
- `[날짜]` → 체크인/체크아웃 날짜
- `[X]박 [X]일` → 실제 숙박일수
- `[금액]` → 실제 예약 금액

### 숙소 정보
- `010-XXXX-XXXX` → 실제 연락처
- `123456-78-901234` → 실제 계좌번호
- `0906번` → 현재 비밀번호 (수시 변경)

---

## 💡 발송 시점

1. **예약 직후** (5분 이내)
   - 웹 예약 완료 시
   - 전화 예약 접수 후
   - 카카오톡 예약 확정 후

2. **입금 확인 후**
   - 별도 "입금 확인 완료" 메시지 발송

3. **체크인 당일 오전**
   - 최종 입실 안내 발송
   - 비밀번호 재확인
