## 📝 문구 커스터마이징 가이드

### ✏️ 수정 필요한 부분
- `[예약자명]` → 실제 예약자 이름
- `[객실명]` → 실제 객실 이름
- `[체크인날짜]` → 실제 체크인 날짜
- `[체크아웃날짜]` → 실제 체크아웃 날짜
- `[금액]` → 실제 금액
- `010-XXXX-XXXX` → 실제 연락처
- `[주소]` → 실제 주소
- `[****]` → 실제 비밀번호/와이파이 정보

### 🎨 이모지 활용 팁
- 📋 예약, 문서
- 💳 결제, 입금
- 🏠 숙소, 집
- 📍 위치, 길찾기
- 🚗 주차, 교통
- ⚠️ 주의사항
- ✅ 확인, 완료
- 🔑 체크인, 키
- 👋 인사
- ⭐ 리뷰, 평가
- 💝 혜택, 선물
- 🌟 추천, 특별

### 💡 작성 팁
1. **간결하게**: 3-4줄 단위로 끊어서 가독성 높이기
2. **구조화**: 이모지로 섹션 구분
3. **필수 정보 우선**: 날짜, 금액, 계좌번호 등
4. **친근한 어투**: 딱딱하지 않게 "~요", "~세요" 사용
5. **CTA 명확히**: "입금해주세요", "연락주세요" 등 행동 유도
