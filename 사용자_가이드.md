## 3. 게스트 예약 방법

### 📱 **모바일/PC 예약 절차**

#### **Step 1: 사이트 접속**
```
1. 브라우저(Chrome/Safari) 열기
2. 주소 입력: accommodation-booking1.vercel.app
3. 자동으로 로그인 페이지로 이동
```

#### **Step 2: 로그인**
```
아이디: guest
비밀번호: guest1234
```

#### **Step 3: 메인 페이지**
```
✓ 아파트 소개 확인
✓ 특징 및 시설 보기
✓ 사진 갤러리 확인
✓ 위치 정보 확인
✓ "예약하기" 버튼 클릭
```

#### **Step 4: 예약 캘린더**
```
1. 캘린더에서 체크인 날짜 클릭 (파란색으로 표시)
2. 체크아웃 날짜 클릭 (범위 선택됨)
3. 일자별 가격 자동 표시
   - 특별 가격이 있는 날짜는 해당 가격
   - 없으면 객실 기본 가격

예시:
✅ 예약가능 (초록색) + 가격
❌ 예약완료 (빨간색)
⏰ 지난날짜 (회색)
```

#### **Step 5: 예약 정보 입력**
```
예약 폼이 자동으로 나타남:
- 객실 선택 (드롭다운)
- 예약자 이름 입력
- 연락처 입력

"예약 완료하기" 버튼 클릭
```

#### **Step 6: 예약 완료**
```
✅ "예약이 완료되었습니다!" 메시지
✅ 관리자 앱에 실시간 알림 전송
✅ 캘린더에 예약 표시
```

## 4. 관리자 앱 설치 방법

### 📱 **Android (Chrome) 설치**

#### **방법 1: 자동 설치 프롬프트 (추천)**
```
1. Chrome 브라우저 열기

2. 주소 입력:
   accommodation-booking1.vercel.app/admin

3. 로그인:
   아이디: admin
   비밀번호: admin1234

4. 대시보드 페이지에서 하단에 파란색 팝업 나타남:
   "앱 설치"
   "홈 화면에 추가하여 빠르게 예약 관리하세요"
   
5. "지금 설치하기" 버튼 클릭

6. 확인 대화상자 → "설치" 클릭

7. 홈 화면에 아이콘 생성됨:
   이름: "속초 아파트 관리자"
   아이콘: 파란색 배경에 '속' 글자
```

#### **방법 2: Chrome 메뉴에서 수동 설치**
```
1. 관리자 페이지 접속 및 로그인

2. Chrome 우측 상단 점 3개 메뉴 (⋮) 탭

3. "앱 설치" 또는 "홈 화면에 추가" 선택

4. "설치" 클릭

5. 홈 화면 아이콘 생성 확인
```

#### **앱 실행**
```
1. 홈 화면에서 "속초 아파트 관리자" 아이콘 탭
2. 앱이 전체 화면으로 실행 (주소창 없음)
3. 로그인 상태 유지 (재로그인 불필요)
```

## 5. 실시간 알림 설정

### 🔔 **알림 활성화 (필수)**

#### **Step 1: 알림 센터 열기**
```
1. 관리자 앱 실행 (또는 브라우저에서 /admin 접속)

2. 우측 상단 벨 아이콘 (🔔) 클릭/탭

3. 알림 센터 패널 열림
```

#### **Step 2: 알림 켜기**
```
1. 패널 상단에 "🔔 알림 켜기" 버튼 확인

2. "🔔 알림 켜기" 버튼 클릭

3. 브라우저 알림 권한 요청 팝업:
   "accommodation-booking1.vercel.app이(가) 
    알림을 보내려고 합니다"

4. "허용" 또는 "Allow" 클릭

5. 상태 변경 확인:
   ✅ "실시간 알림 활성화" (초록색 점 깜박임)
```

#### **Step 3: 테스트 알림**
```
1. 알림 센터에서 "✓ 테스트" 버튼 클릭

2. 화면 상단에 알림 팝업:
   📱 "테스트 알림"
   "알림이 정상적으로 작동합니다! 🎉"

3. 알림 소리 재생 (띵~)

4. 알림 센터에 "테스트 알림" 추가됨

5. 정상 작동 확인!
```

---

### 📱 **실제 예약 알림 테스트**

#### **테스트 시나리오:**
```
[준비물]

- 다른 기기 또는 시크릿 모드

[관리자 앱]
1. 앱 실행
2. 알림 켜기 완료
3. 상태: "실시간 알림 활성화"
4. 앱 열어둔 상태 또는 백그라운드

[게스트 사이트 - 다른 기기/시크릿 모드]
1. accommodation-booking1.vercel.app 접속
2. 로그인 (guest / guest1234)
3. 예약하기 클릭
4. 날짜 선택 및 예약 완료

[관리자 앱]
→ 즉시 푸시 알림 수신! 🎉

알림 내용:
📱 "새 예약 알림"
"홍길동님의 예약이 접수되었습니다.
 객실: 디럭스룸
 체크인: 2026-04-01"

→ 알림 소리 재생
→ 진동 (모바일)
→ 벨 아이콘에 빨간 배지 (1)
→ 알림 센터에 추가
```

### 🔧 **알림 상태 확인**

#### **알림 센터 상태 표시:**
```
✅ 초록색 점 + "실시간 알림 활성화"
   → 정상 작동 중

❌ 빨간색 점 + "알림 차단됨"
   → 브라우저 설정에서 알림 허용 필요

⚠️ 회색 점 + "알림을 켜서 예약 알림 받기"
   → "알림 켜기" 버튼 클릭 필요
```

```
3. "저장"

[비밀번호 변경]
1. 해당 계정 "수정"
2. 새 비밀번호 입력
3. "저장"

[중요]
- 현재 로그인 계정은 삭제 불가
- 비밀번호는 8자 이상 권장
```

#### **게스트 계정**
```
/admin/guest-account

[게스트 계정 수정]
1. 아이디 변경 (기본: guest)
2. 비밀번호 변경
3. 활성화/비활성화 토글
4. "저장"

[주의]
- 단일 게스트 계정 시스템
- 비활성화 시 게스트 로그인 불가
```

## 7. 문제 해결

### ❓ **알림이 안 와요**

#### **체크리스트:**
```
1. 알림 권한 확인
   - 알림 센터 → 상태 확인
   - "실시간 알림 활성화"인지 확인

2. 브라우저 설정 확인
   [Android Chrome]
   설정 → 사이트 설정 → 알림 → 허용
   
   [iOS Safari]
   설정 → Safari → 알림 → 허용

3. 테스트 알림 확인
   - "✓ 테스트" 버튼 클릭
   - 팝업 나타나는지 확인

4. 네트워크 확인
   - WiFi/데이터 연결 확인
   - 페이지 새로고침

5. 앱 재시작
   - 앱 완전 종료
   - 다시 실행
   - 알림 켜기 다시 클릭
```

---

### ❓ **앱 설치 안 돼요**

#### **해결 방법:**
```
[Android Chrome]
1. Chrome 최신 버전 확인
2. 메뉴 → "앱 설치" 직접 선택
3. 또는 "홈 화면에 추가"

[iOS Safari]
1. Safari만 지원 (Chrome X)
2. 공유 버튼 (□↑) 탭
3. "홈 화면에 추가"

[설치 프롬프트 안 나옴]
- 이미 설치되어 있을 수 있음
- 홈 화면 확인
- 브라우저 메뉴에서 수동 설치
```

---

### ❓ **로그인이 안 돼요**

#### **확인 사항:**
```
1. 아이디/비밀번호 재확인
   게스트: guest / guest1234
   관리자: admin / admin1234

2. 대소문자 확인
   (모두 소문자)

3. 공백 확인
   (앞뒤 공백 없이)

4. 계정 활성화 확인
   - 관리자가 비활성화했을 수 있음
   - 관리자에게 문의

5. 캐시 삭제 후 재시도
```

---

### ❓ **예약이 안 돼요**

#### **확인 사항:**
```
1. 날짜 선택 확인
   - 체크인/체크아웃 둘 다 선택했는지
   - 과거 날짜는 선택 불가

2. 객실 선택 확인
   - 드롭다운에서 객실 선택

3. 예약자 정보 입력 확인
   - 이름, 연락처 모두 입력

4. 예약 불가 날짜
   - 빨간색 날짜는 이미 예약됨
   - 다른 날짜 선택

5. 네트워크 확인
   - WiFi/데이터 연결 확인
```

---

### ❓ **사진 업로드가 안 돼요**

#### **해결 방법:**
```
1. 파일 형식 확인
   - JPG, PNG, WEBP 지원
   - GIF는 지원하지 않을 수 있음

2. 파일 크기 확인
   - 최대 10MB 권장
   - 너무 큰 파일은 압축 후 업로드

3. 네트워크 확인
   - 안정적인 WiFi 연결
   - 모바일 데이터 충분한지 확인

4. 브라우저 재시작
   - 페이지 새로고침
   - 캐시 삭제

5. Supabase Storage 확인
   - 관리자에게 문의
   - 저장 공간 확인
```

---

### ❓ **앱이 느려요**

#### **개선 방법:**
```
1. 캐시 정리
   - 브라우저 설정 → 캐시 삭제
   - 앱 재시작

2. 백그라운드 앱 종료
   - 다른 앱 종료
   - 메모리 확보

3. 네트워크 개선
   - WiFi 재연결
   - 라우터 재시작

4. 앱 재설치
   - 앱 삭제
   - 재설치

5. 기기 재시작
   - 기기 완전 재시작
```
